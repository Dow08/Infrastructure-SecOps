# Infrastructure & SecOps

Projets d'infrastructure securisee, deploiement, supervision et detection.
Travaux realises dans le cadre de la formation AIS RNCP37680 et de labs personnels.

## Projets

### nova-syndicate
Infrastructure multi-sites complete — 12 VMs Terraform, 10 playbooks Ansible, SIEM Wazuh, segmentation VLAN, VPN WireGuard, plan de continuite et budget CAPEX/OPEX.
Stack : Terraform, Ansible, Wazuh, WireGuard

### kessel-soc-siem
Deploiement d'un SOC fonctionnel avec Wazuh — integration de sources de logs, tests de detection par simulation de brute force, tuning des regles.
Stack : Wazuh, Linux

### meet-voice-devsecops
Mise en place d'un stack de monitoring complet sur un serveur de reseau social — Prometheus, Grafana, Alertmanager, Loki, Promtail, orchestration Ansible.
Stack : Ansible, Prometheus, Grafana, Loki

### cyber-homelab-vps
Station OSINT auto-hebergee sur VPS — environnement de veille et d'investigation isole.
Stack : Linux, VPS

### vps-osint-hermes
Deploiement One-Shot d'un VPS personnel — outillage OSINT (mini-Palantir) et agent Hermes integre, automatisation complete de l'installation.
Stack : Python, Bash, Linux

### active-directory
Administration Active Directory sous Windows Server — gestion des utilisateurs, GPO, securisation de l'annuaire.
Stack : Windows Server, Active Directory
