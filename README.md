# Infrastructure & SecOps

Projets d'infrastructure sécurisée, déploiement, supervision et détection.
Travaux réalisés dans le cadre de la formation AIS RNCP37680 et de labs personnels.

## Projets

### nova-syndicate
Infrastructure multi-sites complète — 12 VMs Terraform, 10 playbooks Ansible, SIEM Wazuh, segmentation VLAN, VPN WireGuard, plan de continuité et budget CAPEX/OPEX.
Stack : Terraform, Ansible, Wazuh, WireGuard

### kessel-soc-siem
Déploiement d'un SOC fonctionnel avec Wazuh — intégration de sources de logs, tests de détection par simulation de brute force, tuning des règles.
Stack : Wazuh, Linux

### meet-voice-devsecops
Mise en place d'un stack de monitoring complet sur un serveur de réseau social — Prometheus, Grafana, Alertmanager, Loki, Promtail, orchestration Ansible.
Stack : Ansible, Prometheus, Grafana, Loki

### cyber-homelab-vps
Station OSINT auto-hébergée sur VPS — environnement de veille et d'investigation isolé.
Stack : Linux, VPS

### vps-osint-hermes
Déploiement One-Shot d'un VPS personnel — outillage OSINT (mini-Palantir) et agent Hermès intégré, automatisation complète de l'installation.
Stack : Python, Bash, Linux

### active-directory
Administration Active Directory sous Windows Server — gestion des utilisateurs, GPO, sécurisation de l'annuaire.
Stack : Windows Server, Active Directory
